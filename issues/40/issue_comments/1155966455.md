大昔に知ったことなので今更感があるけど、

forkしたリポジトリで作業してると

```
git push
gh pr create -f
```

だと「どのリポジトリにpushする?」とか聞かれて鬱陶しい一方、

```
git push -u
gh pr create -f
```

のように -u オプション等で upstream branchを設定しておくと、聞かれなくて捗る。

ニッチな話だし、みんな知ってるかもだし困ってるの自分ぐらいだろうし、と思ってあえてどこかに書いたことはなかった。
